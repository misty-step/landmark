Ticket 016: First-Run Adoption

Goal Make a new human or agent get useful Landfall output in about a minute without secrets, GitHub coupling, or platform confusion.

Assumptions

  • The zero-secret path should reuse the Rust CLI, especially run --provider local, rather than adding another runtime.
  • The GitHub Action remains an adapter; docs must lead with local CLI and generic CI before Action modes.
  • Setup should recommend one trigger strategy per release family; manual tags should not double-trigger on both tag push and release publish by default.

Acceptance Evidence

  • A replay scenario creates a disposable git repo, runs a zero-secret local preview, and verifies evidence plus release artifacts.
  • check-action-contract rejects stale docs, missing example files, bad model IDs, and Node 24 drift.
  • bin/gate and hosted CI pass, including Linux action binary parity.
